## 🔹 Experiment 4.1 – Employee Management System

A command-line application built using Node.js.

* Add Employee
* List Employees
* Update Employee
* Delete Employee
* Data persistence using JSON file

### Tech Used:

* Node.js
* File System (fs module)

---

## 🔹 Experiment 4.2 – REST API for Card Collection

A RESTful API built using Express.js.

### Endpoints:

* **GET** `/api/cards` → Get all cards
* **POST** `/api/cards` → Add new card
* **PUT** `/api/cards/:id` → Update card
* **DELETE** `/api/cards/:id` → Delete card

### Tech Used:

* Node.js
* Express.js

---

## 🔹 Experiment 4.3 – Booking System

A simple booking API that simulates seat booking.

### Features:

* Book tickets
* Auto-decrease available seats
* Returns booking ID and remaining seats

### Endpoint:

* **POST** `/api/book`

### Tech Used:

* Node.js
* Express.js
